Projects

Here you can find some of the projects I made.

Highlighted projects

gluck-bot

A chatbot built for my school. It is designed to be able to answer questions about our school and the German school system in general. It includes a user system for continuing chats over longer times. Furthermore teachers can use it to share documents with other teachers or students.

TetrOS

Small operating system with bootloader and kernel for playing Tetris. I made this project to learn about the basics of how an OS works under the hood.

Rasterization-Renderer

A basic 3D rasterization render built with C++ and SDL.

IPL

Interpreter for a simple python-like language. It has the goal of making syntax as simple as possible, so you can focus on learning the programming mindset and not syntax.

ILI

Library installer for IPL, aimed at managing dependencies and installation workflows.

Digit-Recognition-Neural-Network

From-scratch neural network for handwritten digit recognition, built to learn how neural networks (and AI for that matter) really work.
